# Лабораторная работа №4: Последовательные одномерные контейнеры

## 🎯 Цели работы
* Изучение алгоритмов формирования и обработки одномерных массивов и последовательных контейнеров (`vector`).
* Программирование и отладка алгоритмов обработки данных.

## 📝 Описание задач (Вариант 8)

### 1. Численное уточнение корней
Реализовано нахождение корня нелинейного уравнения $f(x) = \cos(x + 0.3) - x^2 = 0$ тремя способами:
* Метод бисекции (деления пополам).
* Метод Ньютона (касательных).
* Метод простых итераций.
*Программа выводит таблицу итераций и конечное значение с заданной точностью.*

### 2. Статистика случайных чисел
Генерация последовательности из 1000 случайных элементов, их сортировка и вычисление ключевых статистических показателей (min, max, среднее значение).

### 3. Генератор Фибоначчи
Алгоритм формирования последовательности чисел Фибоначчи заданного размера с использованием контейнера `std::vector`.

### 4. Моделирование систем голосования
Реализация алгоритмов для определения победителя в выборах:
* **Метод Борда** (балльная система на основе рангов).
* **Метод Кондорсе** (система попарного сравнения кандидатов).

### 5. Дилемма заключенного (Теория игр)
Симуляция взаимодействия двух игроков с различными стратегиями:
* `randomChoice` — случайный выбор действия.
* `titForTat` — повторение предыдущего хода противника.
* `grimTrigger` — сотрудничество до первого предательства, после чего всегда следует агрессия.
